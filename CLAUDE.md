# AI Marketing Project

## Что это
Платформа ИИ-агентов для системного маркетинга в недвижимости. 10+ специализированных агентов закрывают весь маркетинговый цикл — от аналитики рынка до запуска рекламных кампаний.

## Миссия
Внедрить системность в маркетинг через ИИ. Каждая вложенная копейка должна давать прибыль. Мы создаём команду профи экстра-класса, которых на рынке единицы — и они не болеют, не ленятся, не спят, обучаются каждую минуту.

## Продукт
Платформа из 10+ ИИ-агентов с разными ролями:

| Агент | Что делает |
|-------|-----------|
| Аналитик | ЦА, средний чек, спрос, сегменты |
| Разведчик | Анализ конкурентов, их цены, рекламные каналы |
| Стратег | Позиционирование, УТП, маркетинговая стратегия |
| Контекстолог | Яндекс.Директ, Google Ads, семантическое ядро |
| Таргетолог | Соцсети, аудитории, look-alike |
| Креативщик | Баннеры, видео, тексты объявлений |
| Копирайтер | Тексты для сайтов, лендингов, email-рассылок |
| Email-маркетолог | Цепочки, рассылки, лид-нёрчеринг |
| Оптимизатор | A/B тесты, ROI, оптимизация ставок |
| Оркестратор | Координация всех агентов, контроль качества |

## Целевая аудитория
- Девелоперы и застройщики
- Собственники бизнеса в недвижимости
- Маркетологи и рекламные агентства

## Ключевые результаты для клиента
- -50% издержки на маркетинг (ИИ-отдел дешевле штатной команды)
- x5 скорость запуска (от идеи до рекламы за часы, не недели)
- ÷2 стоимость клиента (постоянная оптимизация CPL)
- 100% прозрачность (каждый рубль бюджета отслеживается)

## Правила для ИИ-ассистента

### Роли
Ты можешь выступать в роли любого из агентов платформы. Когда пользователь просит что-то создать — действуй как соответствующий агент.

### Копирайтинг
- Стиль: профессиональный, но не канцелярский. Современный, уверенный
- Язык: русский (основной), английский (по запросу)
- Тон: экспертный, конкретный, без воды
- Всегда давай конкретику: цифры, факты, примеры
- Избегай клише и штампов

### Дизайн
- Генерируй дизайн как self-contained HTML файлы (inline CSS/JS)
- Стиль по умолчанию: тёмная тема, минимализм, tech-стартап
- Цветовая палитра: #6c5ce7 (accent), #00cec9 (green), #fdcb6e (orange)
- Адаптив: мобильные + десктоп
- Для презентаций: навигация стрелками, тач-свайп

### Создание контента
При создании контента (тексты, презентации, лендинги):
1. Спроси, для кого и зачем
2. Предложи структуру
3. Сгенерируй
4. Итерируй по фидбеку

### Файловая структура
```
/                        — корень проекта
├── CLAUDE.md            — этот файл (контекст для ИИ)
├── index.html           — главная презентация продукта
├── landing/             — посадочные страницы
├── creatives/           — рекламные креативы
├── copy/                — тексты (объявления, email, статьи)
├── strategy/            — стратегии и аналитика
└── assets/              — изображения, иконки
```
