## Что появилось

В редакторе ПромоСтраниц (апрель 2026) добавлены три группы инструментов для командной работы над статьями - без переключения между сервисами.

## 1. Шеринг черновика до публикации

- Раньше ссылку на статью можно было отправить только после публикации
- Теперь: кнопка «Поделиться» доступна на этапе черновика
- **Режим «Только просмотр»:** получатель видит статью в том виде, в котором её увидит читатель. Авторизация в кабинете не нужна - достаточно ссылки
- **Режим «Просмотр + комментарии»:** статья открывается прямо в редакторе; для оставления комментариев нужна авторизация

## 2. Ветки комментариев прямо в редакторе

- Коллега или клиент выделяет нужный фрагмент текста и оставляет предложение
- Два способа работы с комментариями:
  - Клик по иконкам веток прямо в тексте (счетчик непрочитанных, как в мессенджере)
  - История комментариев - полный список обсуждений, при клике подсвечивается нужный фрагмент

## 3. Сравнение версий

- История версий хранит все состояния статьи с именами авторов и временными метками
- Можно сравнить текущую версию с любой предыдущей (не только с последней)
- Откат к любой прошлой версии - одна кнопка (доступно редакторам и пользователям с доступом в кабинет)

## 4. AI-переписывание фрагментов в режиме сравнения

- Выделить любой фрагмент -> «Улучшить» -> задать AI задачу: переписать короче, добавить примеры, изменить тон
- Результат появляется рядом с исходником
- Можно отменить и попробовать снова

## Полный workflow в одном кабинете

1. AI-генератор готовит черновик статьи по ссылке на продукт (см. `promostranicy-ai-article-generator.md`)
2. Редактирование в редакторе ПромоСтраниц
3. Шеринг черновика клиенту/коллеге для согласования по ссылке
4. Сбор правок через ветки комментариев
5. Запуск статьи после финального одобрения

## Применимость для клиентов KUBRIK

- Упрощает согласование контент-планов и статей с клиентами (застройщиками)
- Больше не нужны отдельные Google Docs для правок - весь процесс внутри кабинета
- AI-переписывание ускоряет доработку статей по фидбеку клиента
- Релевантно при использовании ПромоСтраниц в медиапланах (TOFU/MOFU для недвижимости)
