# Donald Miller — Building a StoryBrand

## Центральная идея

> "Customers aren't looking for another hero — they're looking for a guide."

Клиент = герой. Бренд = наставник (Йода, не Люк Скайуокер).

**Формула:** ПЕРСОНАЖ хочет чего-то → ПРОБЛЕМА → НАСТАВНИК → ПЛАН → ДЕЙСТВИЕ → избегает ПРОВАЛА → УСПЕХ

---

## SB7 Framework — 7 элементов

### 1. CHARACTER — Персонаж (клиент как герой)
- Определить одно центральное желание (не список)
- Категории: финансы, социальные связи, смысл, безопасность, идентичность
- **Aspirational identity:** клиенты покупают версию себя, которой хотят стать

### 2. HAS A PROBLEM — Проблема (три уровня)

**Villain (Злодей):** Персонифицированный антагонист (не конкурент). Четыре свойства: root source, relatable, singular, real.

**External Problem:** Физическая, измеримая. "Мой сайт не конвертирует."
**Internal Problem:** Эмоциональная фрустрация. "Я чувствую себя некомпетентным." — **настоящая причина покупки.**
**Philosophical Problem:** Моральное измерение. "Малый бизнес заслуживает тех же инструментов, что и корпорации."

Идеальный текст затрагивает все три уровня.

### 3. MEETS A GUIDE — Наставник

**Empathy:** "Мы знаем, как это frustrating — тратить деньги и не видеть результата"
**Authority:** Testimonials, Statistics, Awards, Logos — голосом других, не своим

### 4. GIVES A PLAN — План

**Process Plan:** 3 простых шага (оставьте заявку → аудит → стратегия)
**Agreement Plan:** Обещания и гарантии ("Мы никогда не порекомендуем то, что вам не нужно")

### 5. CALLS TO ACTION — Призыв

**Direct CTA:** "Записаться на консультацию" — в правом верхнем углу, после каждого раздела
**Transitional CTA:** "Скачать бесплатный гайд" — для тех, кто не готов

### 6. FAILURE — Провал (ставки)

Без ставок нет истории. Четырёхшаговая формула:
1. Установить уязвимость
2. Объяснить последствия
3. Представить решение
4. Призвать к действию

### 7. SUCCESS — Успех (трансформация)

Три архетипа:
- **Power/Position:** уважение, влияние, признание
- **Union/Wholeness:** тревога уходит, жизнь целостна
- **Self-Realization:** новая версия себя

## One-Liner Formula

```
[ПРОБЛЕМА] + [РЕШЕНИЕ] + [РЕЗУЛЬТАТ]
```

Расширенная:
```
Мы помогаем [кто], которые сталкиваются с [проблема],
[что делаем], чтобы они смогли [результат].
```

---

## BrandScript Template

```
CHARACTER:    Кто герой? Чего хочет?
PROBLEM:      Villain / External / Internal / Philosophical
GUIDE:        Empathy / Authority
PLAN:         3 шага / Agreement
CTA:          Direct / Transitional
FAILURE:      Что будет если не решить?
SUCCESS:      Before / After трансформация
```

---

## Website Wireframe (Above the Fold)

**За 5 секунд посетитель должен понять:**
1. Что вы предлагаете?
2. Как это улучшит мою жизнь?
3. Что мне нужно сделать?

**Структура:** Проблема → Решение → Гид (authority) → Как это работает (план) → Успех (отзывы) → Ставки → CTA

---

## Формулы заголовков

- `[Aspirational identity] — [что им нужно]`
- `Если [проблема] — [решение]`
- `Перестаньте [проблема]. Начните [результат].`
- `[Результат] без [страх/боль]`

---

## Ключевые правила

1. Никогда не начинать с себя — первое слово про клиента
2. Называть проблему до решения
3. Говорить на языке внутренней проблемы
4. CTA везде и всегда
5. Трансформация, а не features
6. **"If you confuse, you lose."**

---

## Диагностика ошибок

| Ошибка | Исправление |
|--------|-------------|
| Бренд как герой | «Вы заслуживаете...» |
| Только внешняя проблема | Добавить internal + philosophical |
| Нет ставок | Добавить failure секцию |
| Нет плана | 3 шага: просто и конкретно |
| CTA спрятан | CTA на каждом экране |
