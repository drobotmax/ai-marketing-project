# Антипаттерны - что НЕ работает

T1 knowledge: при конфликте с курсами (T2) или справкой платформ (T3) - побеждает этот файл.

## [2026-03] Яндекс Директ: ОЗК мертва
**Источник:** Дима + курс Прайса
**Суть:** Оплата за конверсию (ОЗК) в Яндекс Директ практически не работает. После массовых злоупотреблений с фейковыми целями Яндекс ужесточил алгоритм.
**Почему важно:** Справка Яндекса (T3) до сих пор предлагает ОЗК как стратегию. На практике она не зацепится, особенно в поиске.
**Правило для агентов:** НЕ рекомендовать ОЗК в первый месяц запуска. Тестировать можно только как последнее средство, когда другие стратегии не сработали.

## [2026-03] РФ: Google Ads не работает
**Источник:** Дима, Антон
**Суть:** Из-за текущего законодательства РФ работать с Google Ads никто не будет.
**Почему важно:** Модель (T4) и справка Google (T3) предлагают Google Ads как основной канал.
**Правило для агентов:** Для РФ рынка Google Ads НЕ предлагать. Только для международных рынков (Bali, Dubai, Thailand).

## [2026-03] Яндекс Директ: товарный МК не в первый месяц
**Источник:** Дима + курс Прайса
**Суть:** Товарный Мастер кампаний не запускаем в первый месяц. Рассматривать после первого месяца работы, при наличии карточек товаров на сайте.
**Почему важно:** Алгоритм Прайса рекомендует МК на ключевых и авто-аудитории первыми. Товарный МК требует данных.
**Правило для агентов:** Товарный МК включать в план только со второго месяца.

## [2026-03] Ретаргетинг: можно с первого дня
**Источник:** Дима
**Суть:** Ретаргетинг можно запускать хоть на следующий день после старта, если есть оффер с логикой пробития вернувшегося пользователя.
**Почему важно:** Многие шаблоны и курсы рекомендуют ретаргетинг "с 3-й недели". Это неверно - привязывать надо к наличию оффера и логики, а не к таймлайну.
**Правило для агентов:** НЕ привязывать ретаргетинг к таймлайну. Запускать когда есть оффер.

## [2026-03] VK лид-формы: минимум полей
**Источник:** Дима
**Суть:** Чем меньше полей в лид-форме, тем больше конверсий. Минимальная форма: имя + телефон.
**Правило для агентов:** Рекомендовать короткие формы (2-3 поля) для cold аудитории. Расширенные формы (5 полей) - только для квалификации, с пониманием что CPL вырастет.

## [2026-03] Недвижимость: коллтрекинг обязателен
**Источник:** Дима, Real Estate Playbook
**Суть:** Для недвижимости 40-60% лидов приходят через звонки. Без коллтрекинга теряем половину данных для оптимизации.
**Правило для агентов:** Коллтрекинг - BLOCKER для запуска. Не запускать кампании без него.

## [2026-03] Яндекс Директ: обучать на квалифицированных лидах
**Источник:** Дима, Real Estate Playbook
**Суть:** Оптимизация по сырому lead volume быстро портит качество трафика. Нужно обучать кабинет на квалифицированных лидах и продажах через CRM feedback.
**Почему важно:** Дешевый лид может быть бесполезным. Цикл сделки в недвижимости 30-180 дней.
**Правило для агентов:** Планировать передачу офлайн-конверсий из CRM в рекламный кабинет. Разделять лиды и квалифицированные лиды как разные сигналы оптимизации.
