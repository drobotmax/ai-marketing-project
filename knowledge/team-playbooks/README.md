# Team Playbooks - T1 Knowledge (highest priority)

Практический опыт команды KUBRIK. Этот слой имеет **высший приоритет** в иерархии знаний:
T1 (этот слой) > T2 (курсы/книги) > T3 (справка платформ) > T4 (знания модели).

При конфликте с курсами или справкой платформ - побеждает то, что здесь.

## Как добавлять

После реальной кампании, инсайта от Димы/Антона или обнаружения антипаттерна:

```markdown
## [YYYY-MM-DD] Инсайт: [название]
**Источник:** Дима / Антон / результаты кампании X
**Суть:** [что узнали]
**Почему важно:** [контекст]
**Правило для агентов:** [конкретная инструкция]
```

## Файлы

- `anti-patterns.md` - что НЕ работает (антипаттерны)
- `campaign-results/` - результаты реальных кампаний (CPL, конверсии, что сработало)
- По мере роста: `dima-insights.md`, `anton-insights.md`
