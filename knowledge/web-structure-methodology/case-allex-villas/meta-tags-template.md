# Шаблоны мета-тегов (Allex Villas, English market)

Источник — Google Sheet «Мета-теги alexvillasgroup» (см. LINKS.md). Шаблоны для девелоперского сайта на международном рынке.

## Принципы написания

1. **Title:** 50–60 символов, начинается с целевого ключа, заканчивается брендом.
2. **Description:** 150–160 символов, обещание + 1 факт + CTA.
3. **Бренд + год основания** в title главной = trust-сигнал для YMYL.
4. **ROI / число% / число лет** в description = конкретика, повышает CTR.
5. **Локация в title** для всех страниц по объектам — критично для гео-SEO.

```

```

**Переменные:**
- `[Название_Проекта]` — имя проекта (Villa Sky, Aqua Residences, …)
- `[Тип_Объекта]` — Villa / Apartment / Land Plot / Hotel
- `[Локация]` — Seminyak / Canggu / Ubud / Uluwatu / Tabanan / …
- `[Ключевое_УТП]` — beachfront / panoramic ocean view / passive income / …

## Что менять под рынок

| Рынок | Цифра ROI в title | Особенности |
|---|---|---|
| Бали (Allex Villas) | 16% | freehold-нюанс для иностранцев |
| Дубай | 8–12% | Golden Visa-нюанс |
| Сочи / Краснодар | 10–14% | акцент на «инвест-апартаменты» |
| Алматы | 6–10% | акцент на «жилая + аренда» |
| Тайланд (Пхукет) | 7–10% | freehold/leasehold |

Цифра ROI должна быть **обоснована** (не выдумана) — берётся из реальных проектов клиента или из отраслевых отчётов.
