## E-E-A-T-чеклист (YMYL-режим)

Поскольку недвижимость = деньги, Google применяет YMYL-фильтры. Что проверяем у каждого конкурента:

- ✅ **Экспертность:** есть ли блог с глубокими статьями?
- ✅ **Авторитетность:** упоминания в СМИ? ссылки с авторитетных сайтов?
- ✅ **Доверие:**
  - чёткие контакты,
  - инфо о компании,
  - лицензии,
  - отзывы,
  - портфолио реализованных проектов.

## Главные выводы из анализа (для применения на любом девелопере)

1. **Микроразметка обязательна.** У всех 10 конкурентов schema.org стоит. Если у клиента нет — это первый таск разработке.
2. **Блог = экспертность.** У всех ranking-конкурентов (Traffic > 1000) есть блог с глубокими статьями. Без блога нельзя.
3. **Авторы статей.** Половина конкурентов проигрывает на отсутствии авторов — это **окно возможности**: сделаем своих экспертов с фото+ФИО+соцсетями = выиграем по E-E-A-T.
4. **Отзывы.** Почти у всех сильных — отзывы (хотя бы цитатами). Без отзывов = слабый E-E-A-T.
5. **Core Web Vitals — слабая сторона ниши.** Большинство конкурентов медленные. Это окно: оптимизированный по PSI сайт обгоняет нишу.
6. **Mobile-first.** >60% трафика недвижимости — мобильный. Десктоп-only сайт мёртв.
7. **AI Visibility = новая ось.** 0–35 у конкурентов. Это новый фронт для GEO-оптимизации (см. навык seo-geo).

## Расширение анализа: смежная ниша «управление недвижимостью»

Когда у клиента в услугах есть «управляющая компания» или «сопровождение инвесторов», нужен **второй раунд анализа** под управление-нишу:
- balimanagement.villas
- betterplace.cc
- impianbali.com
- elldproperty.com
- friede-property.com
- nagisa-bali.com
- villabalimanagement.com

Это разная ЦА (собственник vs инвестор-покупатель) и разные кластеры запросов («management in bali» = 43 запроса). Структура и семантика расширяется под этот трек.

## Шаблон применения для нового клиента

1. Собрать список 8–12 прямых конкурентов с региона/ниши клиента.
2. Прогнать через Ahrefs (Age, DR, Traffic, KW, RD).
3. Прогнать через PSI (mobile/desktop).
4. Прогнать вручную: schema.org, AI-упоминания, E-E-A-T-чеклист.
5. Посчитать **медиану** по нише — целевой ориентир.
6. Найти **окна возможностей**: где конкуренты слабы (например, отзывы, авторы блога, скорость).
7. Сделать **пресет рекомендаций**: что внедрить на сайт клиента, чтобы войти в топ-3 ниши за 6–12 месяцев.
