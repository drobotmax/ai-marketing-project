## Модуль 1: Что такое SEO

### Определение

SEO (Search Engine Optimization - оптимизация под поисковые системы) - это набор методов и мер, которые призваны повысить значимость сайта в глазах поисковых систем для увеличения поискового трафика. Простыми словами - это набор задач, выполнив которые, ваш сайт получит большую вероятность попасть в ТОП-10 выдачи поисковых систем, а значит приводить больше пользователей.

### Органическая выдача

Органическая выдача - это список сайтов, которые сформировала поисковая система по введенному запросу в строке поиска на основе алгоритма.

### SEO vs Контекстная реклама (Директ)

| Критерий | SEO | Контекст (Директ) |
|----------|-----|-------------------|
| Трафик | Много трафика | Ограниченный бюджетом |
| Сайт | Развивает сайт | Не влияет на сайт |
| Лиды | Теплые клиенты | Клиенты сразу |
| Скорость | Долгосрочный результат | Результат сразу |
| Оценка | Сложнее понять уровень специалиста | Проще понять |
| Бюджет | Фиксированная оплата | Зависит от кликов |

Опытные предприниматели используют оба канала параллельно.

### Факторы ранжирования

Поисковые системы используют ИИ и более 800 факторов для определения позиций.

Ключевые факторы:
- **Качество контента** - уникальный, полезный, релевантный
- **Структура сайта** - для Yandex и Google
- **Коммерческие факторы** - цены, контакты, доверие
- **Бренд** - узнаваемость, упоминания
- **Скорость загрузки** - Core Web Vitals
- **Поведенческие факторы (ПФ)** - важны для Yandex; веса - для Google

### 3 ключевых фактора продвижения
1. Структура сайта (Yandex, Google)
2. Текстовая релевантность
3. ПФ для Yandex, веса (ссылки) для Google

### Каким сайтам SEO не подойдет
- Одностраничные сайты (лендинги)
- Монобренды
- Стартапы без сформированного спроса

### 5 мифов о SEO
1. "Выйду в ТОП без SEO" - не выйдете
2. "Справимся своими силами" - нужна экспертиза
3. "Новый сайт продвинуть невозможно" - возможно
4. "SEO уже не работает" - работает
5. "Никто не дает гарантии" - неправда

## Модуль 2: Семантика

### Что такое семантическое ядро

Семантическое ядро (СЯ, "семантика") - это набор ключевых запросов, которые описывают сайт и то, что на нем размещается. Семантическое ядро есть у каждой страницы на сайте. Из всех ключевых слов формируется большое СЯ сайта.

Ключевой запрос ("ключ") - это обычные запросы, которые пользователи вбивают в поисковик, когда ищут что-то в интернете.

### Классификация запросов

По типу выдачи:
- **Коммерческие** - с намерением покупки ("купить септик")
- **Информационные** - поиск информации ("как установить септик")

По другим признакам:
- **Геозависимые / геонезависимые**
- **Сезонные / несезонные**
- **Конкурентные / неконкурентные**

По частотности:
- **ВЧ** - высокочастотные
- **СЧ** - среднечастотные
- **НЧ** - низкочастотные

### Маркерные запросы

Маркером (или маркерным запросом) называют слово или словосочетание, наиболее точно отражающее суть конкретной страницы сайта. Обычно в качестве основного "маркерного" запроса для страницы берется содержимое заголовка h1. У одной страницы может быть несколько маркерных запросов.

Как найти маркерные запросы:
- Анализ конкурентов
- Мозговой штурм
- Яндекс Метрика, Google Analytics
- Название разделов и категорий
- Сбор на основе mind-карты

### Процесс формирования СЯ

1. **Сбор маркерных запросов** - основа структуры
2. **Сбор ключевых запросов из Wordstat** (wordstat.yandex.ru)
3. **Сбор поисковых подсказок** из различных источников (в программе KeyCollector)
4. **Сбор частотности** - статистика, как часто набирают запросы
5. **Кластеризация запросов** - группировка по темам

### Сбор СЯ вручную

Инструменты:
- wordstat.yandex.ru - основной источник
- Плагин WordStater - для удобного сбора
- Левая колонка - прямые запросы, правая - "Люди ищут" (LSI запросы)
- LSI запросы - синонимы и связанные ключевые слова для расширения семантики

Операторы частотности в Wordstat:
- **Общая** - широкое соответствие
- **"Точная-фразовая"** - в кавычках
- **"!Абсолютная"** - с восклицательным знаком

### Сбор СЯ автоматом

Сервисы:
- bukvarix.com
- KeyCollector (KK)
- Key.so
- И другие

### Кластеризация

Кластеризация ключевых слов - это процесс объединения ключевых слов в группы (кластеры) по общим характеристикам, значению или темам. Получившиеся группы используют в SEO-продвижении - размещают на целевых страницах, чтобы продвигать их в поисковых системах.

Сервисы кластеризации: KeyCollector (KK), Just Magic, Пиксель тулс и др.

После кластеризации: формируются ТЗ для копирайтера, пишутся заголовки h1, мета Title, мета description.

### Мониторинг позиций

Сервисы: Топвизор, Мегаиндекс, Key.so

## Модуль 3: Внутренняя оптимизация

### Предварительные факторы (до начала работ)

- **Домен** - выбор, возраст, история
- **CMS** - система управления контентом
- **Хостинг** - надежность, скорость
- **Скорость загрузки** - Core Web Vitals
- **Протокол HTTPS** - обязательно SSL-сертификат

### Настройка аналитики

- Добавление сайта в Яндекс Вебмастер и Google Search Console
- Подключение Яндекс Метрики
- Правильное подтверждение прав на сайт

### Техническая оптимизация

- **Код страницы** - из чего состоит контент сайта
- **Адаптивная верстка** - мобильная версия
- **Интерактивные элементы** - фильтры, видео, слайдеры должны работать корректно
- **URL** - понятные, ЧПУ
- **Битые ссылки и 301 редиректы** - устранение
- **robots.txt** - настройка индексации
- **sitemap.xml** - карта сайта
- **Canonical** - каноническая ссылка для дублей
- **Сжатие изображений** - оптимизация веса
- **Ошибки HTML и CSS** - устранение
- **Микроразметка** - Schema.org для rich snippets

### Заголовки и мета-теги

Соответствие Title и h1 ключевому слову - один из самых важных факторов ранжирования.

Элементы:
- **Title** - заголовок страницы в поисковой выдаче
- **h1** - основной заголовок на странице
- **Description** - описание для поисковой выдачи
- **h2-h6** - подзаголовки для структурирования контента

### Навигация и структура сайта

- **Расширение структуры** - для охвата семантики и расширенного ответа на запрос пользователя
- **Анализ конкурентов** - изучение их структуры
- **Категории и подкатегории** - логическое дерево сайта
- **Хлебные крошки** - навигация на всех внутренних страницах
- **Коммерческие страницы** - О компании, Доставка, Оплата и т.д.

### Работа с контентом

- Какой контент нужен: сплошной текст vs каталог vs страница услуги
- Какие блоки размещать: **сквозные** (на всех страницах) и **уникальные** (специфичные)
- **Текстовая релевантность** - анализ на основе медианы конкурентов (20 самых популярных слов)
- Заказ текста у копирайтера по ТЗ

### Юзабилити и конверсия

- Функциональные элементы: фильтр, сортировка, калькулятор, избранное, сравнение, эквайринг
- Контакты и способы связи
- Формы заявки
- Квиз и интерактивные элементы
- Качественные изображения и дизайн

## Модуль 4: Внешняя оптимизация

### Ссылочная масса

Ссылочная масса - это совокупность всех активных внешних ссылок, размещенных на сторонних ресурсах и ведущих на оптимизируемый сайт.

Метрики ссылочной массы:
- **Trust Rank** - доверие
- **Domain Rank** - авторитетность домена
- **Org vs SEO** - соотношение органических и SEO-ссылок

### Методы наращивания ссылочной массы

1. **Регистрация в каталогах** - базовые ссылки
2. **Покупка вечных ссылок на биржах** - постоянные ссылки
3. **Покупка крауд-ссылок** - ссылки на форумах и в комментариях
4. **Ссылки в социальных сетях** - профили и посты
5. **Статьи на открытых площадках** - VC.ru, Дзен
6. **Покупка ссылок напрямую у вебмастера** - прямые договоренности

### Ведение социальных сетей

У компаний, следящих за своими соц. сетями, результат в SEO обычно лучше при прочих равных.

Каналы: Telegram, VK, VC.ru, YouTube

### Репутация компании в сети

Рейтинг компании на Яндекс Картах, наличие отзывов - это важный фактор ранжирования для Яндекса.

- Регистрация в Яндекс Бизнес (95%+ заполнение)
- Google My Business
- Сбор отзывов на площадках

## Модуль 5: Результат в SEO

### Связь позиций и посещаемости

Чем выше позиция, тем больше кликов. Поэтому растет посещаемость.

На первом этапе важно собирать запросы всех типов: НЧ, СЧ, ВЧ.

### Как правильно смотреть аналитику

- **Обновления сервисов** - алгоритмы поисковиков обновляются
- **Атрибуции** - правильная модель атрибуции в метрике
- **Легальные манипуляции** - допустимые методы
- **Сезонный бизнес** - учитывать сезонность при оценке

### Видимость сайта

Видимость сайта - это метрика в SEO, которая показывает, какой процент целевой аудитории увидел ссылку на ваш сайт в поисковой выдаче.

С помощью видимости сайта специалист отслеживает динамику изменений (снижения и роста) и оценивает эффективность SEO-продвижения.

Сервисы: Топвизор, Мегаиндекс, Key.so

## Модуль 6: Взаимодействие клиента и SEO-студии

### Стоимость SEO

- За что платит клиент: работа специалиста, контент, ссылки, инструменты
- Тарифы зависят от сложности ниши и объема работ

### Сроки достижения результата

Зависят от конкуренции. Авторитетные SEO-эксперты рекомендуют белое SEO.

**Примерный roadmap на 12 месяцев:**

| Период | Результат |
|--------|-----------|
| 1-2 мес | Сайт преображается, начинает соответствовать требованиям поисковиков |
| 3-4 мес | Вырастут позиции по СЯ, начнет расти посещаемость, первые заявки |
| 4-6 мес | Трафик сильно вырастет, по многим запросам ТОП-10 |
| 6-12 мес | Трафик, позиции, заявки продолжат расти, значительная доля рынка |

Кто обещает результат за 2 месяца - скорее всего лукавит.

### Что после окончания SEO-продвижения

Рекомендации по поддержке:
1. **Периодическое обновление страниц** - раз в 2-3 месяца: редактирование текста, новые блоки, формы
2. **Репутация** - новые отзывы на Яндексе и других сервисах
3. **Информационные статьи** - регулярный блог
4. **Ссылки** - при публикации в соцсетях всегда ставить ссылку на продвигаемые страницы

## Модуль 7: Отчетность в SEO

### Виды отчетов

"Три кита" отчетности:
1. **Ежедневный Google-отчет** - план работ, KPI, чек-лист
2. **Гостевая ссылка на позиции** - Топвизор для клиента
3. **Ежемесячный отчет** - полная сводка

### Что должно быть в отчете

- Клиенты приходят за трафиком и посещаемостью, а говорят о заявках
- Ежемесячные обновления с понятными метриками
- Высокий уровень сервиса в коммуникации

## Модуль 9: Продажи SEO-услуг

### Поиск клиентов по шагам

1. Определить нишу и состав услуг
2. Провести конкурентный анализ
3. Развивать компетенции фрилансера

**Три составляющих успешного специалиста:**
- **Технические навыки**: маркетинговое мышление, знание интерфейсов, Excel, автоматизация
- **Система работы**: воронка поиска заказчиков, воронка удержания, структуры и скрипты, юридические документы, регламенты
- **Уверенность и самопрезентация**: умение продать себя, самооценка, понимание задач заказчика

> Клиент покупает подтвержденный результат

### Рекламные каналы

- Протестировано более 10 каналов
- ТОП-3 канала: 2 условно бесплатных + 1 платный

### Как продавать SEO

- Книга: "Гениальные скрипты продаж" - Михаил Гребенюк
- Собирать обратную связь: от текущих клиентов, купивших и не купивших
- Продавать результат, кейсы, сервис, качество
- Всех переводить на **КЭВ** (ключевой этап воронки)

### Воронка продаж SEO

1. **Звонок + бриф** - первый контакт, квалификация
2. **Понять, ваш клиент или нет** - фильтрация
3. **Предложить аудит (1 КЭВ)** - рассказать, что будет в аудите
4. **Запросить доступы** - метрика, вебмастер
5. **Презентация (2 КЭВ)** - аудит + КП + рассказ об отчетах и планах
6. **Проговорить следующий шаг** - четкий call-to-action
7. **Договор, счет** - закрытие сделки

### 5 документов для продаж
1. Бриф первого контакта
2. Аудит сайта (шаблон)
3. Коммерческое предложение (КП)
4. Оффер
5. Конкурентный анализ

## Приложение A: Чек-лист коммерческих факторов сайта

### Общесайтовые факторы (20 пунктов)
1. Наличие и конкурентоспособность цен на сайте
2. Наличие SSL-сертификата (HTTPS)
3. Широта ассортимента (матрица товаров и услуг)
4. E-mail на своем домене
5. Сайт зарегистрирован в Яндекс Бизнесе (заполнение 95%+)
6. Сайт зарегистрирован в Google My Business
7. Есть категории и подкатегории (дерево сайта)
8. Mobile friendly версия сайта
9. Кликабельные телефон и почта на 1 экране
10. Наличие номера 8800
11. Наличие городского номера с кодом региона
12. Удобная навигация (хлебные крошки)
13. Активные группы в соц. сетях с виджетами на сайте
14. Страница Политика конфиденциальности
15. Чек-бокс "Согласен на обработку персональных данных" во всех формах
16. Инфографика на основных посадочных страницах
17. Онлайн-консультант / виджет обратного звонка
18. Ссылки на мессенджеры (Telegram, WhatsApp, Viber)
19. Корректно работающий поиск
20. Отсутствие внешней рекламы

16. Сотрудники
17. Политика конфиденциальности
18. Новости компании
19. Блог с информационными статьями

### Шапка сайта (11 пунктов)
1. Логотип и подстрочник (2-5 слов о деятельности)
2. Номер телефона с кодом города (для ИМ - 8800)

4. Адрес офиса/филиалов
5. Режим работы

7. Ссылки на соцсети (VK, OK)
8. Кнопка обратного звонка
9. Иконка поиска
10. Для ИМ: корзина, избранное, сравнение
11. Выбор города (если несколько регионов, номер меняется по IP)

1. Насыщенный подвал с гиперссылками на важные разделы
2. Логотип + описание деятельности
