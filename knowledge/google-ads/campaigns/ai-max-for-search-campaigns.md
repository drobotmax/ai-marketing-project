# AI Max for Search Campaigns

AI Max is Google's next-generation search campaign type, replacing Dynamic Search Ads. As of April 2026, AI Max is out of beta and generally available (GA).

## Key Metrics

- Average +7% more conversions or conversion value at similar CPA/ROAS vs. using only search term matching (non-Retail advertisers, full feature suite)
- Hundreds of thousands of global advertisers already scaling with AI Max

## Three Core Features

1. **Search term matching** - AI expands beyond keywords using real-time intent signals + website content; finds untapped queries
2. **Text customization** - Generates relevant headlines based on website content (replaces DSA headline generation)
3. **Final URL expansion** - Sends users to the most relevant page on the site (controllable)

All three features enabled together deliver the +7% lift. Enable full suite for maximum performance.

## Enhanced Controls

- **Brand controls** - restrict or allow specific brand terms in AI expansion
- **Location controls** - geographic precision steering
- **Text guidelines** - constrain AI-generated copy (critical for premium/brand-sensitive segments)

## DSA Migration Timeline

### Phase 1 (April - August 2026): Voluntary upgrades

- Upgrade tools rolling out now for DSA users (port historical settings + data to standard ad groups)
- ACA and broad match users: upgrade via pop-up banner in Google Ads UI
- Recommended: upgrade now to keep full control over setup

### Phase 2 (September 2026): Automatic upgrades

- Remaining eligible campaigns auto-upgrade to AI Max
- New DSA creation ends (Google Ads UI, Editor, API)
- Notifications sent to accounts before transition
- Migration mirrors legacy settings:
  - DSA: dynamic ad groups → standard ad groups; all 3 AI Max features enabled; legacy URL controls preserved
  - ACA: search term matching + text customization enabled by default
  - Broad match: search term matching enabled by default
- All upgrades complete by end of September 2026

## DSA vs AI Max Comparison

| Feature | DSA | AI Max |
|---------|-----|--------|
| Query expansion | Landing page data | Landing page + real-time intent signals |
| Headline generation | Based on website | AI-generated with text guidelines |
| URL selection | Dynamic | Final URL expansion (controllable) |
| Brand controls | Limited | Yes |
| Location controls | Standard | Enhanced |
| Status | Deprecated (Sep 2026) | GA (April 2026) |

## When to Use

- As a catch-all expansion layer beyond keyword-based campaigns
- When DSA was previously used (mandatory migration)
- For campaigns using ACA or campaign-level broad match
- **For premium/brand-sensitive segments**: enable text guidelines to constrain AI-generated copy and preserve brand voice

## Implementation Notes

- Enable all 3 features for maximum +7% lift
- Use brand exclusions to protect brand campaigns from AI Max expansion
- One-click experiments available to test performance impact before full rollout
- After upgrade: monitor search term reports, adjust URL expansion exclusions as needed
