## Что такое Ads Advisor

Ads Advisor - AI-агент внутри Google Ads (на базе Gemini). Используется для получения инсайтов, оптимизации кампаний и теперь - автоматизации рутинных задач по безопасности аккаунта.

Доступен: всем аккаунтам с интерфейсом на английском языке глобально. Другие языки - в процессе раскатки.

**Статус новых функций:** выходят в ближайшие месяцы (announced 2026-04-21).

## 3 новые функции безопасности

### 1. Proactive Troubleshooting (Проактивный поиск нарушений)

- **Что:** Real-Time Policy Reviews - мгновенная проверка политики при создании и редактировании кампаний
- **Поведение:** Ads Advisor сам сканирует аккаунт и сайт при возникновении проблем, не ждет запроса
- **Что дает:** четкий путь к устранению нарушения + подтверждение исправления перед подачей аппеляции
- **Применимость для KUBRIK:** помогает избежать отклонений объявлений, особенно в тематике недвижимости (special category)

### 2. 24/7 Security Monitoring (Мониторинг безопасности)

- **Что:** непрерывный мониторинг аккаунта с персонализированными рекомендациями
- **Включает:** аудит пользователей, флаг на неактивные аккаунты и подозрительные домены
- **Новый инструмент:** security insights dashboard - показывает текущий уровень защиты
- **Команда:** `show me my security level` - snapshot статуса безопасности
- **Пароли:** Ads Advisor внедряет passkeys в Google Ads (замена паролей + 2FA)

### 3. Instant Certifications (Мгновенная сертификация)

- **Что:** автоматизация процесса получения сертификатов для рекламы в регулируемых категориях
- **Как:** Ads Advisor сам определяет, нужна ли сертификация по отрасли и стране
- **Процесс:** автовыдача сертификата или подача заявки в 1 клик (с одобрения рекламодателя)
- **Применимость:** актуально для недвижимости, финансов, медицины - категорий с обязательной верификацией

## Применимость для клиентов KUBRIK

- **Недвижимость (international):** Proactive Troubleshooting критичен - Housing category в Google Ads требует сертификации
- **Мониторинг:** настроить security dashboard при запуске любого нового Google Ads аккаунта
- **Certifications:** при работе с real estate на международных рынках - использовать Instant Certifications для ускорения запуска
