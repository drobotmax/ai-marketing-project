# Media Plan: ЖК "Парковый" - Горизонт Девелопмент

## Общие параметры
- Бюджет: 500 000 руб/мес (hard constraint)
- Гео: Краснодар + край (основное), Москва/СПб (релоканты)
- Период: месяц 1 (тестовый запуск, 30 дней)
- Цель: 80-120 лидов/мес при CPL до 5 000 руб `[assumption]`
- Тип объекта: первичная недвижимость, комфорт+, 280 квартир
- Google Ads: НЕ включен (РФ рынок)

## Approved Channel Plans

### Channel 1: Яндекс Директ
- **Owner:** Контекстолог
- **Objective:** Demand capture (поиск) + demand creation (МК, РСЯ) + ретаргетинг
- **Approved budget:** 225 000 руб/мес (195K committed daily + 30K масштабирование)
- **Why now:** Основной канал лидогенерации для РФ первички. Горячий интент через поиск, МК для разведки аудиторий
- **Key dependencies:**
  - Яндекс Метрика - **BLOCKER**, не запускать без неё
  - Коллтрекинг - **BLOCKER**, 40-60% лидов через звонки
  - SEO-теги сайта - нужны для корректной работы МК
  - AmoCRM -> Метрика - offline conversion feedback
- **Success signal для Phase 2:** CPL < 5 000 руб `[assumption]`, 30+ лидов за первый месяц, qualified rate > 25%
- **Included specialist components:**
  - 8 кампаний: 2x МК, 4x Поиск (бренд, комнатность, ипотека, конкуренты), 1x РСЯ, 1x Ретаргетинг
  - Семантическое ядро: 6 кластеров (бренд, комнатность, ипотека, конкуренты, гео, релоканты)
  - Объявления: 3 модели (AIDA, ODC, 4U) по каждому интенту
  - Минус-слова: базовый список + cross-group negatives
- **Deferred / cut items:**
  - Товарный МК - месяц 2+ (нужны данные и обученный кабинет)
  - Смарт-баннеры - месяц 2+ (догоняющий формат по фиду)
  - Гео Москва/СПб для релокантов через РСЯ - месяц 2+ (сначала VK тестирует)
  - Автостратегии - после набора достаточного объема конверсий (30+ за 30 дней)
  - Google Ads - не ранее месяца 3, только как optional experiment

### Channel 2: VK Ads
- **Owner:** Таргетолог
- **Objective:** Lead generation через лид-формы (cold) + ретаргетинг
- **Approved budget:** 100 000 руб/мес
- **Why now:** Быстрые заявки через лид-формы, минимальное трение. VK - единственный social-канал в РФ
- **Key dependencies:**
  - VK Pixel на сайте - нужен для ретаргетинга и LAL
  - Интеграция VK Lead Ads -> AmoCRM - без неё ручной перенос лидов
  - Креативы: рендеры парка, визуализации, планировки
  - Парсинг конкурентов (TargetHunter) - для аудитории подписчиков ЮСИ/AVA/ССК
- **Success signal для Phase 2:** CPL < 4 000 руб `[assumption]`, qualified rate > 20%, 25+ лидов за первый месяц
- **Included specialist components:**
  - 4 кампании: Cold семьи (50K), Cold инвесторы (25K), Cold релоканты (15K), Ретаргетинг (10K)
  - Audience matrix: 10 аудиторий (3 cold, 3 warm, 1 hot, 3 medium)
  - Experiment plan: 5 тестов (аудитории, офферы, форматы, длина формы, гео релокантов)
  - Optimization rules по сигналам (CPM, CTR, CPL, frequency, CRM feedback)
- **Deferred / cut items:**
  - Look-alike аудитории - Phase 2 (после 100+ лидов)
  - Масштабирование ретаргетинга до 15-20% - только при подтвержденных результатах cold

### Channel 3: Классифайды (Авито + Циан + Яндекс.Недвижимость)
- **Owner:** Контекстолог / менеджер по размещению
- **Objective:** Demand capture - прямой поиск покупателями на площадках
- **Approved budget:** 125 000 руб/мес (Авито 50K + Циан 50K + Я.Недвижимость 25K)
- **Why now:** Обязательный канал для РФ первички. Покупатели активно ищут объекты на классифайдах
- **Key dependencies:**
  - Актуальные карточки объектов с ценами и планировками
  - XML-фид (есть по брифу)
  - Регулярное обновление карточек (цены, наличие)
- **Success signal:** Стабильный поток входящих запросов с площадок, CPL на уровне рынка
- **Included components:**
  - Размещение карточек объектов на всех трех площадках
  - Платное продвижение карточек
- **Deferred / cut items:**
  - Авито Ads (баннерная реклама) - из резерва, месяц 2+

### Channel 4: VK-сообщество (контент)
- **Owner:** Таргетолог + SMM
- **Objective:** Trust, прогрев, доверие для тех, кто видел рекламу
- **Approved budget:** 25 000 руб/мес (контент-продакшн + минимальное продвижение постов)
- **Why now:** Без живого сообщества лид-формы VK работают "в пустоту" - пользователь видит пустой профиль и не доверяет
- **Key dependencies:**
  - Фото/видео хода стройки от застройщика
  - Рендеры парка и комплекса
  - SMM-менеджер или подрядчик на ведение
- **Success signal:** 500+ целевых подписчиков за 3 месяца, engagement rate > 3%, вовлеченные автоматически попадают в ретаргетинг
- **Included components:**
  - 7 рубрик: ход стройки, парк и среда, район, ипотека, планировки, Q&A, отзывы
  - Pinned post с основным оффером
  - Paid sync: вовлеченные -> аудитория ретаргетинга

### Channel 5: Резерв на тесты
- **Owner:** Planner (арбитраж)
- **Approved budget:** 25 000 руб/мес
- **Правило:** Не тратить в месяц 1 без данных. Активировать в месяце 2+ на основании результатов Phase 1
- **Потенциальные направления:** Telegram Ads, Авито Ads, новые гипотезы
- **Trigger активации:** Phase 1 каналы стабилизированы, есть понимание CPL и qualified rate

## Cross-Channel Optimization Rules

1. **Каннибализация:** VK и Яндекс атакуют разные стадии воронки (VK = cold/demand creation, Яндекс Поиск = hot/demand capture). Overlap минимален. Мониторить через UTM: если пользователь приходит из обоих каналов, атрибуция по last click, но отслеживать assisted conversions
2. **Sales capacity:** Если ОП не успевает обрабатывать лиды (speed-to-lead > 30 мин) - НЕ масштабировать VK cold, пока не решена проблема на стороне продаж
3. **CPL divergence:** Если CPL VK < 2 000, а Яндекс Поиск > 6 000 через 14 дней - не перекидывать бюджет автоматически. Поиск дает более горячий спрос, проверить qualified rate прежде чем резать
4. **Ретаргетинг overlap:** VK и Яндекс ретаргетинг работают на разных пулах (VK pixel vs Метрика). Если пользователь в обоих пулах - не проблема, разные каналы, разные touchpoints
5. **Классифайды:** Стабильный канал, не реагировать на колебания CPL в платных каналах перераспределением классифайдного бюджета
6. **CRM reconciliation:** Еженедельно сверять лиды из всех каналов с CRM. Дешевый CPL без qualification loop не является аргументом для scale
