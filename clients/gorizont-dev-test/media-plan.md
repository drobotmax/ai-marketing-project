# Медиаплан: ЖК "Парковый" - Горизонт Девелопмент

## Общие параметры

- **Бюджет:** 500 000 руб/мес (~$5 500)
- **Гео:** Краснодар + край (primary), Москва + СПб (релоканты)
- **Период:** 3 месяца (месяц 1 - запуск и тесты, месяц 2 - расширение, месяц 3 - оптимизация)
- **Цель:** 80-120 лидов/мес при CPL до 5 000 руб
- **Объект:** ЖК "Парковый", комфорт+ класс, 280 квартир, от 4.2 млн руб, сдача Q4 2027
- **Позиционирование:** "Парк у дома - не обещание, а 3 гектара вашего двора"

## Яндекс Директ - 175 000 руб/мес

### Measurement Setup (до запуска)

1. **Яндекс Метрика** - счётчик + цели: form_submit, phone_click, whatsapp_click, plan_view, thanks_page
2. **Колтрекинг** (Calltouch / CoMagic) - 3 подменных номера (поиск, РСЯ, ремаркетинг)
3. **GTM** - единый контейнер
4. **UTM-разметка** - `utm_source=yandex&utm_medium=cpc&utm_campaign={campaign_name}&utm_content={ad_id}&utm_term={keyword}`
5. **CRM интеграция** - передача статусов лидов (мин. ручная выгрузка 2 раза/неделю)

## Optimization Rules

### Decision Matrix

| Сигнал | Действие |
|--------|----------|
| CTR < 3% (поиск Яндекс) | Ревизия заголовков, проверить релевантность ключ-объявление |
| CTR < 0.8% (VK) | Пересмотреть hook в креативе, проверить offer-audience match |
| CTR < 0.3% (РСЯ) | Обновить креативы, проверить площадки, сузить аудитории |
| CPM > 400 руб (VK) | Расширить аудиторию, проверить частоту, обновить креативы |
| CPL ок, но качество лидов плохое | Сузить семантику / усложнить форму, проверить landing page |
| CPL высокий, CTR высокий | Проблема конверсии сайта - тестировать новый лендинг |
| CPL высокий, CTR низкий | Проблема объявлений - переписать тексты, проверить ключи |
| Frequency > 3.0 за 7 дней (VK) | Обновить креативы, расширить аудиторию |
| Бюджет исчерпывается до 14:00 | Увеличить бюджет или сузить таргетинг |
| Дозвон < 50% | Speed-to-lead risk - SLA < 15 мин, авто-сообщение |
| Конверсии Метрика != CRM (> 20%) | Проверить трекинг, дубли, спам-заявки |

### Ритм оптимизации

- **Ежедневно (недели 1-2):** Чистка поисковых запросов, мониторинг CTR/CPM, проверка модерации, чистка площадок РСЯ
- **Еженедельно:** Анализ CPL по кампаниям, корректировки ставок (устройства, гео, время), обновление минус-слов, сверка Метрика vs CRM
- **Ежемесячно:** Полный аудит waste spend, пересмотр бюджетов между каналами, обновление объявлений, отчёт клиенту

## Risks & Dependencies

| Риск | Вероятность | Влияние | Митигация |
|------|-------------|---------|-----------|
| Трекинг не настроен | Высокая | Критическое | BLOCKER - не запускать без проверенных целей и тестовых конверсий |
| CRM интеграция отсутствует | Высокая | Высокое | Минимум: ручная выгрузка 2 раза/неделю. Без этого нет feedback loop |
| Speed-to-lead > 30 мин | Средняя | Высокое | SLA на перезвон < 15 мин, авто-сообщение в VK при получении лида |
| Landing page с CR < 2% | Средняя | Высокое | A/B тест лендинга неделя 2-3. Мин. требования: скорость < 3с, mobile, форма above the fold |
