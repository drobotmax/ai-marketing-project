## 1. Measurement Setup

### Текущий статус
- GA4: нет | Goals: не настроены
- GTM: нет
- Метрика: нет | Цели: не настроены
- Колтрекинг: нет
- CRM integration: нет
- Offline conversion import: нет

### Что нужно настроить (до запуска)

1. **Яндекс Метрика** - установить счётчик на сайт, настроить цели:
   - JS-событие `form_submit` - отправка формы обратной связи / заявки
   - JS-событие `phone_click` - клик по номеру телефона
   - JS-событие `whatsapp_click` - клик по кнопке WhatsApp
   - Составная цель `plan_view` - просмотр страницы планировок (глубина + время)
   - URL-цель `thanks_page` - посещение страницы "Спасибо"
   - Микроцель `scroll_50` - прокрутка 50% страницы
   - Микроцель `time_on_site_60s` - 60+ секунд на сайте
2. **Колтрекинг** (Calltouch / CoMagic / Callibri) - подменные номера для Яндекс Директ, минимум 3 номера (поиск, РСЯ, ремаркетинг)
3. **GTM** - единый контейнер для управления тегами Метрики, колтрекинга, пикселей ретаргетинга
4. **GA4** (опционально, для будущего масштабирования на Google Ads) - события, аудитории для ремаркетинга
5. **CRM интеграция** - передача статусов лидов обратно в Метрику (offline conversions). Минимум: ручная выгрузка 2 раза в неделю. Целевая: автоматическая через API
6. **UTM-разметка** - единая система: `utm_source=yandex&utm_medium=cpc&utm_campaign={campaign_name}&utm_content={ad_id}&utm_term={keyword}`

### Ключевые конверсии для оптимизации

| Event | Type | Ценность | Платформа |
|-------|------|----------|-----------|
| form_submit | Macro | 5 000 руб | Метрика + Яндекс Директ |
| phone_call (30с+) | Macro | 5 000 руб | Колтрекинг + Метрика |
| whatsapp_click | Macro | 3 000 руб | Метрика |
| plan_view | Micro | 500 руб | Метрика |
| scroll_50 | Micro | - | Метрика |
| time_on_site_60s | Micro | - | Метрика |

**Важно:** Макро-конверсии (form_submit, phone_call) используются для оптимизации ставок. Микро-конверсии - для анализа качества трафика и поведения.

### Мастер кампаний - настройки

**МК 1: На ключевых словах**
- Ключи: топ-10 горячих фраз из кластеров 1-2
- Заголовок: "Квартира с парком от 4.2 млн - ЖК Парковый"
- Текст: "Центральный район Краснодара. Парк 3 га во дворе. Ипотека 0.1%. Сдача Q4 2027"
- Изображения: 5 шт (рендеры парка, фасад, двор, планировка, район на карте)
- Стратегия: максимум конверсий, CPA не выше 5 000 руб
- Бюджет: 2 500 руб/день

**МК 2: На авто-аудитории**
- Без ключей - алгоритм Яндекса подбирает аудиторию по содержимому сайта
- Тот же креатив что в МК 1
- Стратегия: максимум конверсий, CPA не выше 5 500 руб
- Бюджет: 1 500 руб/день
- Примечание: дать 3-5 дней на обучение, не трогать

**Товарный МК (ручной фид)**
- Фид: XML с карточками квартир (студии, 1к, 2к, 3к) - цена, площадь, изображение планировки
- Алгоритм сам генерирует объявления из фида
- Стратегия: максимум конверсий
- Бюджет: 1 500 руб/день
- Запуск: неделя 2 (после сбора первичной статистики с МК)

## 7. Optimization Rules

### Ежедневно (первые 2 недели)

- Чистка поисковых запросов в поисковых кампаниях (каждые 2-3 дня минимум)
- Проверка статуса показа всех объявлений (модерация Яндекса 1-3 дня)
- Мониторинг дневного расхода - не должен исчерпываться до 14:00
- Проверка работоспособности целей в Метрике (тестовая конверсия)
- Проверка корректности колтрекинга (тестовый звонок)
- Мониторинг CTR по группам - ниже 3% на поиске = тревога
- Чистка площадок в РСЯ (отключить площадки с CTR > 1% и 0 конверсий после 500+ показов)

### Еженедельно

- Анализ CPL по кампаниям / группам / ключевым словам
- Корректировки ставок:
  - По устройствам (обычно mobile конвертирует хуже для недвижимости - корректировка -10-30%)
  - По гео (Краснодар город vs край - разная конверсия)
  - По времени суток (на основе данных - обычно пик 10-14 и 18-21)
  - По полу/возрасту (после накопления данных)
- Обновление минус-слов из отчёта по поисковым запросам
- Проверка Quality Score / CTR / позиции показа
- Анализ площадок РСЯ - отключение мусорных
- Сверка количества лидов в Метрике с CRM (расхождение > 20% = проблема трекинга)

### Ежемесячно

- Полный аудит: waste spend (ключи/площадки без конверсий), top performers, возможности расширения
- Решение о переходе на автостратегии (оплата за конверсию) при 10+ конверсий/неделю
- Обновление текстов объявлений - новые углы, сезонные офферы, актуальные цены
- Синхронизация с CRM: quality lead rate, cost per qualified lead, конверсия лид->просмотр->бронь
- Пересмотр бюджетов между кампаниями на основе CPL и качества лидов
- Расширение или сужение семантики на основе данных
- Отчёт для клиента: лиды, CPL, расход, качество, рекомендации

### Decision Matrix

| Сигнал | Действие |
|--------|----------|
| CTR < 3% (Поиск Яндекс) | Ревизия заголовков, проверить релевантность ключ-объявление, проверить позицию |
| CTR < 0.3% (РСЯ) | Обновить креативы, проверить площадки, сузить аудитории |
| CPC растёт, CTR стабильный | Проверить аукцион (новые конкуренты?), расширить семантику, тестировать новые углы |
| CPL ок, но качество лидов плохое | Сузить семантику, добавить минус-слова, проверить landing page, проверить квалификацию ОП |
| CPL высокий, CTR высокий | Проблема в конверсии сайта - тестировать новый лендинг, формы, CTA |
| CPL высокий, CTR низкий | Проблема в объявлениях или релевантности - переписать тексты, проверить ключи |
| Бюджет исчерпывается до 14:00 | Увеличить бюджет или сузить таргетинг (гео, время, ставки), разнести по времени |
| МК в обучении > 7 дней | Проверить объём данных, упростить цель конверсии, увеличить бюджет |
| Конверсии в Метрике != CRM | Проверить трекинг, дубли, спам-заявки, настроить фильтрацию |
| Конкурент запустил акцию | Обновить объявления, добавить свой оффер, усилить ретаргетинг |
| Сезонный спад (лето, НГ каникулы) | Снизить бюджет на 20-30%, усилить ретаргетинг, подготовить контент для прогрева |

## 8. Risks & Dependencies

| Риск | Вероятность | Влияние | Митигация |
|------|-------------|---------|-----------|
| Трекинг не настроен или работает некорректно | Высокая | Критическое | BLOCKER - не запускать кампании без проверенных целей. Тестовые конверсии до старта |
| Landing page с низкой конверсией (< 2%) | Средняя | Высокое | A/B тест лендинга на неделе 2-3. Минимальные требования: скорость < 3с, мобильная версия, форма above the fold |
| Высокая конкуренция в аукционе (CPС выше прогноза) | Высокая | Среднее | Начать с точного соответствия, расширять постепенно. Усилить бренд-кампании (дешёвые клики) |
| Сезонность - летний спад (июнь-август) | Средняя | Среднее | Заложить снижение бюджета на 20-30% летом, компенсировать ретаргетингом |
| Отсутствие CRM feedback loop | Высокая | Высокое | Минимум: ручная выгрузка статусов 2 раза/неделю. Без этого оптимизация по качеству невозможна |
| Модерация Яндекса отклоняет объявления | Средняя | Среднее | Подготовить 2-3 варианта объявлений на группу. Не использовать superlatives, проверить документы застройщика |
| Каннибализация между кампаниями | Средняя | Среднее | Кросс-групповые минус-слова, приоритет по типу кампании (бренд > гео > общие) |
| Мусорный трафик из РСЯ | Высокая | Среднее | Жёсткая чистка площадок первые 2 недели. Бюджет РСЯ - не более 15% от общего |
