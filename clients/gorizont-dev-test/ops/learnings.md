# Learnings

## Wins
- 2026-03-31: pipeline смог собрать целостную стратегию под реальный девелоперский бриф, а не абстрактный маркетинговый кейс
- 2026-04-04: позиционирование объекта хорошо держится на одном сильном механизме - парк 3 га во дворе

## Failures / Gaps
- 2026-03-31: launch blockers по measurement были известны, но не оформлены как отдельная operational memory
- 2026-04-04: без Метрики и коллтрекинга performance-оптимизация остается частично слепой
- 2026-04-04: privacy / consent слой есть в брифе, но не подтвержден отдельным audit artifact

## Repeated Patterns
- Для недвижимости readiness по аналитике и звонкам нельзя оставлять "на потом" - это влияет на всю систему оптимизации
- Сильнее всего работают конкретные продуктовые механизмы, а не общие обещания комфорта

## Skill Updates Needed
- Отдельный launch-readiness skill / checklist для объектов недвижимости
- Отдельный session discipline для strategy, sales QA и launch audits

## KB Updates Needed
