# Встречайте Yandex Mobile Ads SDK 8 — минимум времени на запуск монетизации, максимум дохода
Рекламная сеть Яндекса выпустила Yandex Mobile Ads SDK 8 — новую версию SDK для монетизации мобильных приложений. Теперь SDK стал ещё легче и проще в использовании. Рассказываем, что изменилось.
## Что нового в Yandex Mobile Ads SDK 8
### SDK теперь на 30% легче*
Чем компактнее SDK, тем быстрее скачивается и меньше весит ваше приложение. А значит, выше конверсия в установку.
* Размер SDK 8 уменьшен по сравнению с SDK 7. В релизах 8.x мы продолжим снижать размер и повышать стабильность SDK
### Современный DevEx и API: выпускайте фичи, а не ловите баги
Меньше кода, меньше отладки — больше времени на то, что действительно важно.
- Миграция с помощью ИИ
Автоматизировали переход на новую версию с помощью скила для ИИ-агентов. Он берёт на себя всю работу с документацией: анализирует код и адаптирует его под API версии 8. Всего 15 минут* — и проект готов.
* По данным наших замеров
- Код, который приятно писать
SDK поддерживает Kotlin Coroutines и Swift async/await — рекламу можно загружать линейно и предсказуемо.
- Генератор идентификаторов SKAdNetwork
Разработали скрипт для автогенерации идентификаторов SKAdNetwork в Info.plist. Корректная атрибуция конверсий из показа в установку означает, что вы не упускаете доход.
### От интеграции до первых денег — за минимум времени
Подключайте рекламу ещё быстрее благодаря готовым инструментам и современным фреймворкам.
- Декларативные UI-фреймворки
Используйте готовые декларативные компоненты под SwiftUI и Jetpack Compose, которые ведут себя как часть вашего кода. Меньше кода — меньше ошибок в вёрстке.
- Compose Multiplatform
Разрабатываете приложения для Android и iOS на Compose Multiplatform? Cоздавайте интерфейс рекламных блоков без дублирования вёрстки. Единая кодовая база — минимум рутины и быстрый запуск.
- InStream API
Выберите, что нужно вашему проекту: быстрый запуск InStream-рекламы с использованием встроенного плеера или интеграция собственного плеера с полным контролем над предзагрузкой и запуском каждого рекламного блока.
- CTV API
Зарабатывайте на больших экранах: API для видеорекламы на Android TV открывает новый источник дохода.
### Профессиональный Unity Kit
Теперь вы можете настраивать рекламу прямо в Unity Editor, а также автоматически проверять проект перед релизом — запускайте монетизацию быстрее и снижайте риск ошибок после публикации.
- Unity Package Manager
Автоматическое управление версиями SDK и зависимостями снижает риск ошибок интеграции и обеспечивает стабильную работу рекламы и атрибуции.
- Unity Integration Manager
Мастер настройки автоматически проверяет конфигурацию SDK и параметры интеграции, помогая выявить возможные ошибки перед запуском.
- Unity UI Inspector
Встроенный инструмент для поиска и исправления проблем с рекламой. Выявляйте ошибки до релиза, чтобы избежать потери дохода и обеспечить стабильный пользовательский опыт.
