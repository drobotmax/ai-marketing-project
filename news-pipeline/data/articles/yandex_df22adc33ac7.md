# В редакторе ПромоСтраниц появились инструменты для совместной работы
Согласовывать статьи стало проще: делитесь черновиком по ссылке, собирайте правки в ветках комментариев и сравнивайте любые версии между собой. Всё это — внутри кабинета, без переключения между сервисами.
## Делитесь статьёй до публикации
Раньше ссылку на статью можно было отправить только после публикации. Теперь это доступно на этапе черновика. Нажмите «Поделиться» и настройте доступ:
- Только просмотр
- Просмотр с возможностью оставлять комментарии
Если открыт только просмотр, получатель увидит статью такой, какой её увидит читатель. Авторизация в кабинете ПромоСтраниц не нужна — достаточно ссылки.
Если открыт доступ с комментариями, статья откроется прямо в редакторе.

## Обсуждайте текст прямо в редакторе
Если нужно не просто показать статью, но и получить обратную связь, включите доступ для комментирования. Коллега или клиент перейдёт по ссылке, выделит нужный фрагмент и оставит предложение. Для этого потребуется только авторизоваться.
Работать с комментариями можно двумя способами:
- Кликать по иконкам веток прямо в тексте — рядом с каждой отображается счётчик непрочитанных сообщений, как в мессенджере.
- Открыть историю комментариев и просмотреть все обсуждения. При клике на ветку редактор подсветит нужный фрагмент текста.
Редактировать статью через такую ссылку нельзя — только смотреть и комментировать.

## Сравнивайте любые версии между собой
История версий хранит все состояния статьи с именами авторов и временем изменений. Сравнить текущую версию можно не только с предыдущей, но и с любой другой из списка.
Например, коллега, который согласовывал статью две недели назад, может открыть историю версий по той же ссылке, найти свой вариант и увидеть, что именно изменилось.
Если нужно вернуться к любой из прошлых версий, это можно сделать одной кнопкой. Возможность доступна редакторам и пользователям с доступом в кабинет.

## Переписывайте фрагменты с помощью AI
В режиме сравнения выделите любой фрагмент, нажмите «Улучшить» и дайте AI задачу: переписать короче, добавить примеры, изменить тон. Результат появится рядом с исходником. Если он вам не подойдёт, можно отменить и попробовать снова.

## Весь цикл работы — в одном кабинете
AI-генератор готовит черновик статьи по ссылке на ваш продукт. Вы дорабатываете текст в редакторе, отправляете на согласование по ссылке, собираете правки в комментариях и запускаете статью после финального одобрения. Благодаря этим инструментам создать материал с нуля можно внутри одного кабинета.
