# Мобильное приложение Директа: новые возможности для специалистов
Теперь в приложении можно воспользоваться расширенной аналитикой в новом Мастере отчётов и ИИ-помощнике, авторизоваться через управляющий аккаунт и работать с рекомендациями.
Мы продолжаем развивать возможности мобильного приложения Директа, чтобы с его помощью управлять кампаниями и анализировать результаты продвижения могли как предприниматели, так и специалисты. Рассказываем, какие инструменты для опытных пользователей уже доступны в приложении.
## Углублённая аналитика в новом Мастере отчётов
Помимо сводной статистики по вашим кампаниям, которая доступна в разделе «Обзор», в мобильном приложении теперь можно использовать и расширенную аналитику в новом Мастере отчётов.
### Сценарии работы со статистикой в приложении Директа, которые будут полезны специалистам
- Сохранённые отчёты
Если вы привыкли пользоваться кастомными отчётами и смотреть нужные показатели по кампаниям, теперь они всегда под рукой во вкладке «Мои отчёты» в мобильном приложении Директа.
- Библиотека отчётов
Набор готовых отчётов под различные аналитические задачи и типы кампаний тоже стал доступен в приложении.
- ИИ-помощник
Теперь можно не только уточнить цифры и статистику по продвижению в ИИ-помощнике, но и сразу перейти в отчёт в новом Мастере отчётов. Задайте вопрос — и в один клик откройте нужный отчёт с уже подобранными группировками, метриками и фильтрами. Если нет времени анализировать отчёт, помощник суммаризирует данные.
- Самостоятельное создание отчёта
В мобильном приложении доступны и другие привычные настройки из нового Мастера отчётов: группировки, метрики, фильтры, выбор периода, режим сравнения, цели, атрибуции и работа с графиками. Вы можете использовать их для создания новых отчётов.
### Как перейти к отчётам в мобильном приложении
- Откройте приложение и кликните по иконке аккаунта в правом нижнем углу.
- В блоке «Инструменты» перейдите в раздел «Статистика».
- Выберите подходящий отчёт из вкладки «Мои отчёты» или из Библиотеки либо создайте новый по клику по «+» в правом нижнем углу.
В открывшемся отчёте можно внести изменения: скорректировать настройки, период, включить режим сравнения, настроить детализацию данных и так далее.

### Советы по работе с отчётом в приложении
- Чтобы таблица с отчётом открылась на весь экран, кликните кнопку «Развернуть». Настройки отчёта при этом скроются. Вернуть их можно с помощью кнопки «Свернуть».
- Используйте настройку «Плотность отображения данных». Она поможет показать больше данных в таблице.
- Закрепите столбец с группировкой, чтобы при горизонтальной прокрутке было удобно смотреть большие таблицы.
- Используйте фильтры, чтобы проанализировать определённый сегмент трафика. Фильтры по группировкам находятся в настройках отчёта, а фильтры по метрикам — в столбце с метрикой в отчёте.

## Управляющий аккаунт
В мобильном приложении Директа появилась возможность авторизоваться через управляющий аккаунт и быстро переключаться между доступными аккаунтами. Эта функция будет полезна специалистам по продвижению, которые работают с несколькими аккаунтами.
Кликните по логину в правом нижнем углу и в списке доступных аккаунтов выберите тот, который хотите открыть.

## Оптимизация кампаний с помощью рекомендаций
Повысить эффективность продвижения помогут рекомендации, которые теперь доступны и в мобильном приложении. Директ подскажет, какие изменения внести в настройки кампаний, чтобы получать больше кликов и конверсий.
С помощью рекомендаций можно за несколько кликов оптимизировать кампанию: изменить бюджет, цену клика или конверсии, скорректировать цели продвижения, добавить объявления и многое другое.

Перейти на страницу с рекомендациями можно через нижнее меню. При клике по иконке раздела отобразится список рекомендаций — останется применить нужные, чтобы внести изменения в настройки кампаний.
Подробнее о рекомендациях в Директе читайте в Справке.
### Управляйте продвижением с помощью мобильного приложения
- Запускайте кампании через Простой старт, Мастер кампаний или ИИ-помощника
- Пополняйте кабинет и управляйте настройками бюджета
- Редактируйте кампании, которые уже доступны в аккаунте, и управляйте ими
- Создавайте лендинги и используйте их в продвижении
- Анализируйте продвижение как с помощью простых дашбордов, так и с помощью расширенной статистики из нового Мастера отчётов
- Используйте рекомендации Директа, чтобы повысить эффективность кампаний
