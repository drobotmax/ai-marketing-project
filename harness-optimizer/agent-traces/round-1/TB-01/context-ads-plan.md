## 7. Optimization Rules

### Ежедневно (первые 3 недели)
- Чистка поисковых запросов (добавление минус-слов)
- Чистка площадок РСЯ (отключение мусорных сайтов)
- Проверка расхода бюджета (не исчерпывается ли к обеду)
- Мониторинг статуса показа объявлений

### Еженедельно
- Анализ CPL по кампаниям, группам, ключам
- Корректировки: устройства, время суток, гео
- Обновление минус-слов
- Проверка CTR по группам (если < 3% на поиске - ревизия)

### Ежемесячно
- Полный аудит: waste spend, top performers, expansion opportunities
- Синхронизация с CRM: qualified lead rate, cost per qualified lead
- Решение о подключении Мастер кампаний (месяц 2)
- Решение о подключении Товарного МК (месяц 2-3, нужен фид)

### Decision Matrix
| Сигнал | Действие |
|--------|----------|
| CTR < 3% (Поиск) | Ревизия заголовков, проверить релевантность ключ -> объявление |
| CPL > 3,000 руб на кампании | Проверить CR сайта, сузить семантику, усилить минус-слова |
| CPL ок, но qualified rate < 30% | Сузить семантику, проверить лендинг, добавить квалифицирующие элементы |
| Budget exhaustion до 14:00 | Сузить гео или время показа, пересмотреть ставки |
| РСЯ CPL > 2x поиск | Чистка площадок, пересмотр креативов, снижение бюджета |

## 8. Risks & Dependencies

- **Tracking:** Метрика и колтрекинг должны быть настроены ДО запуска - blocker
- **Landing page:** gorizont-dev.ru должен иметь рабочую форму, быстрый лендинг, корректную мобильную версию
- **Конкуренция в аукционе:** Краснодар - высококонкурентный рынок, CPC может быть выше прогноза
- **Сезонность:** апрель-май - сезон активного спроса, благоприятное время для запуска
- **CRM feedback loop:** без обратной связи по качеству лидов оптимизация будет слепой
- **Ипотечные программы:** необходимо уточнить актуальные условия для корректных объявлений
