## 8. Risks & Dependencies

- **Метрика не настроена** - без целей оптимизация слепая [KB: Прайс, Step 0] - BLOCKER
- **Landing page quality** - gorizont-dev.ru должен корректно работать на мобильных, быстро загружаться [KB: contextologist skill - если landing слабый, это blocker]
- **CRM feedback loop** - без офлайн-конверсий кабинет обучается на сырых лидах, качество падает [KB: primary-real-estate-direct-playbook, п.9 - оптимизация по lead volume портит качество]
- **Конкуренция в аукционе** - крупные застройщики (ЮгСтройИмпериал и др.) могут перебивать ставки [Assumption]
- **Сезонность** - весна = пик спроса, хорошее время для запуска [KB: contextologist skill - весна/осень пики]
- **Колтрекинг** - без него 30-50% конверсий (звонки) невидимы [KB: primary-real-estate-direct-playbook, п.7]
